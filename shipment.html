Shipment plugin

My own take on a shop/market system
Over 5,000 lines of code and 250 lines of configurability!

About the project

The project was initially started for free for a server called TreeGrounds, however, as the project expanded it becaome closed source.
Over a month of work went into this project, with many revisions of the GUI's until I was perfectly happy with their looks.
Overall, this project taught me a lot about working with clients to build a project to meet their specificication and then going above and beyond.
Furthermore, I learnt a lot about working with Spigot's Conversation API, a topic I had previously little experience working with.

Graphical User Interfaces

The plugin has many GUI's, allowing admin's to configure the shops in game via an assortment of menus
The menu featured to the left is the main setup menu, displaying the previous shipments (red),
and current/upcoming shipments (yellow). Clicking on a shipment allows you to edit the content for sale.
